# Tomorrow's Earth Expo 2026 - Brand Guidelines

## Brand Identity

**Tomorrow's Earth Expo 2026** is a global youth innovation platform dedicated to sustainability, environmental responsibility, and forward-thinking technology. The brand represents a movement of young innovators working together to create a sustainable future.

## Animations & Transitions

### Timing
- Fast: 150ms (micro-interactions)
- Standard: 200ms (button hover, fade)
- Slow: 300ms (page transitions)

### Easing Functions
- Ease In: cubic-bezier(0.4, 0, 1, 1)
- Ease Out: cubic-bezier(0, 0, 0.2, 1)
- Ease In-Out: cubic-bezier(0.4, 0, 0.2, 1)
- Spring: cubic-bezier(0.34, 1.56, 0.64, 1)

### Common Animations
- **Fade In:** Opacity 0 → 1 over 300ms
- **Slide Up:** Translate Y 30px → 0 over 300ms
- **Scale In:** Scale 0.95 → 1 over 200ms
- **Hover Lift:** Translate Y -4px on hover

## Logo Variations

### Light Background (Standard)
- Use full-color logo with Deep Blue and Emerald Green
- Ensure minimum 20px clear space

### Dark Background
- Use white or light variant of logo
- Maintain contrast ratio of at least 4.5:1

### Monochrome (Black)
- Use single-color black version
- For print or limited-color applications

### Monochrome (White)
- Use single-color white version
- For dark backgrounds

## Do's and Don'ts

### Do's ✅
- Use the official logo files provided
- Maintain clear space around the logo
- Use the correct color palette
- Follow typography hierarchy
- Use consistent spacing
- Apply shadows for depth
- Test contrast ratios for accessibility
- Use the compressed WebP versions for web

### Don'ts ❌
- Don't distort or stretch the logo
- Don't change the colors without approval
- Don't add effects (gradients, patterns) to the logo
- Don't use outdated logo versions
- Don't mix fonts inconsistently
- Don't use colors outside the approved palette
- Don't create text with insufficient contrast
- Don't use the logo without clear space

## Implementation Notes

### Web Usage
- Always use the compressed WebP versions for faster page loads
- Provide PNG fallbacks for older browsers
- Use responsive image techniques for different screen sizes
- Optimize images with proper alt text and titles

### Print Usage
- Use the original PNG versions for highest quality
- Ensure CMYK color conversion for print
- Maintain minimum 300 DPI resolution
- Test color accuracy with print vendor

### Social Media
- Use the icon version for profile pictures (512x512px minimum)
- Use the main logo for cover images
- Maintain brand consistency across all platforms
- Follow platform-specific sizing guidelines
